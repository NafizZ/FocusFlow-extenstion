# Privacy Policy

Last updated: 2026-01-19

FocusFlow is a browser extension that helps users stay focused with timers, break reminders, and optional site blocking. This policy explains how data is handled.

## Data Collection
FocusFlow does not collect, transmit, sell, or share personal data. There are no analytics, trackers, or remote servers used by the extension.

## Local Storage
The extension stores the following data locally on your device using browser storage:
- Focus timer settings and session state
- Block list entries (default and custom)
- Daily focus and distraction statistics

This data never leaves your device.

## Permissions
FocusFlow uses the following permissions to provide its core functionality:
- Storage: save settings, session state, and daily stats locally
- Tabs: detect the active tab URL to apply blocking
- Alarms: keep the timer accurate in the background
- Notifications: show break reminders
- Host access (http/https): check the current page URL and show a soft-block overlay

## Data Retention and Deletion
All data is stored locally and can be removed by clearing extension data or uninstalling the extension.

## Changes
If this policy changes, the updated version will be published in this repository.
